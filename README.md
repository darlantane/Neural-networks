# Neural Networks - Learning and Implementation Projects

This repository contains a collection of Jupyter notebooks dedicated to learning and practical implementation of neural networks and deep learning. The projects cover fundamental concepts as well as advanced architectures.

## Educational Objectives

This repository serves as an educational portfolio covering:
- Fundamental concepts of neural networks
- Practical implementation with Python/Jupyter
- Applied projects on real datasets
- Modern architectures (RNN, Transformers, Word2Vec)

## Technologies Used

- **Language**: Python (100% Jupyter Notebooks)
- **Libraries**: Likely NumPy, Pandas, Scikit-learn, TensorFlow/PyTorch
- **Environment**: Jupyter Notebook/Lab
